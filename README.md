# NYC Taxi Dashboard 2025

Interactive dashboard analyzing 44.2M yellow taxi trips in NYC for 2025.

## Features
- Choropleth map of 262 taxi zones
- Congestion pricing impact analysis
- Hourly and weekly demand patterns
- Borough-level statistics
- Data quality report (9.4% invalid trips removed)

## Data Source
NYC TLC Trip Record Data (yellow taxis, Jan-Dec 2025)
